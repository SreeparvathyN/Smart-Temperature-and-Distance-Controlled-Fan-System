# Smart Temperature and Distance Controlled Fan System

A microcontroller-based smart cooling system that automatically controls a fan based on surrounding temperature and object proximity. The system uses a temperature sensor to monitor real-time conditions and an ultrasonic sensor to detect presence, adjusting the fan and servo movement accordingly.

---

## 📌 Project Overview

The **Smart Temperature Controlled Fan System** is built using Arduino and basic electronic components. It demonstrates how automation can be used to regulate environmental comfort efficiently without manual intervention.

In this project:

* A temperature sensor continuously reads ambient temperature
* An ultrasonic sensor detects object distance (presence detection)
* Arduino processes both inputs together
* A DC fan is controlled based on temperature and distance conditions
* A servo motor provides mechanical movement based on system activation

---

## ⚙️ Features

* Automatic fan ON/OFF based on temperature and distance
* Object/presence-based activation using ultrasonic sensor
* Real-time sensor monitoring
* Adjustable threshold values in code
* Servo motor oscillation during active cooling
* LCD display support
* Simulation-ready design

---

##  Components Used

* Arduino UNO
* Temperature Sensor (NTC Thermistor in hardware / TMP36 used in simulation)
* Ultrasonic Sensor (HC-SR04)
* DC Motor + Fan blade
* Servo Motor

* Jumper wires

---

## 🔌 Circuit Design

* The temperature sensor reads ambient values and sends analog data to Arduino.
* The ultrasonic sensor measures distance by sending and receiving sound pulses.
* Arduino evaluates both temperature and distance conditions together.
* If conditions are satisfied:

  * DC fan is activated via transistor switching
  * Servo motor oscillates to indicate active cooling mode
* If conditions are not met:

  * Fan remains OFF
  * Servo returns to default position

### Important Notes:

* TMP36 is used in Tinkercad simulation since thermistor is not available there.
* Ultrasonic sensor (HC-SR04) is used to add presence-based activation.
* 1N4007 diode is connected across the motor in reverse bias:

  * Stripe side (cathode) → Positive motor terminal
  * Protects circuit from back EMF generated by motor

---

## ⚙️ Working Principle

1. Temperature sensor continuously measures ambient conditions
2. Ultrasonic sensor checks object distance in front of system
3. Arduino reads both sensor values
4. If:

   * Temperature exceeds threshold AND
   * Distance is within defined range (2 cm to 20 cm)
5. Then:

   * Fan turns ON
   * Servo oscillates to show active mode
6. Otherwise:

   * Fan remains OFF
   * Servo resets to neutral position

## 💻 Code

```cpp
#include <Servo.h>
#include <LiquidCrystal.h>

int fanPin = 9;
int trigPin = 3;
int echoPin = 4;

long duration;
int distance;

Servo myServo;
int servoAngle = 90;
int direction = 1;

LiquidCrystal lcd(12,11,5,6,7,8);

void setup()
{
  Serial.begin(9600);

  pinMode(fanPin, OUTPUT);

  pinMode(trigPin, OUTPUT);
  pinMode(echoPin, INPUT);

  myServo.attach(10);

  lcd.begin(16,2);
  lcd.print("Smart Fan");
  delay(2000);
  lcd.clear();
}

void loop()
{
  int temperature = analogRead(A0);

  digitalWrite(trigPin, LOW);
  delayMicroseconds(2);

  digitalWrite(trigPin, HIGH);
  delayMicroseconds(10);

  digitalWrite(trigPin, LOW);

  duration = pulseIn(echoPin, HIGH);

  distance = duration * 0.034 / 2;

  Serial.print("Temp Sensor: ");
  Serial.print(temperature);
  Serial.print("  Distance: ");
  Serial.println(distance);

  lcd.setCursor(0, 0);
  lcd.print("T:");
  lcd.print(temperature);
  lcd.print(" D:");
  lcd.print(distance);
  lcd.print("   ");

  if (temperature > 535 && distance > 2 && distance < 20)
  {
    analogWrite(fanPin, 255);

    lcd.setCursor(0, 1);
    lcd.print("Cooling ON    ");

    myServo.write(servoAngle);

    servoAngle += direction;

    if (servoAngle >= 150) direction = -1;
    if (servoAngle <= 30) direction = 1;

    delay(15);
  }
  else
  {
    digitalWrite(fanPin, LOW);

    myServo.write(90);

    lcd.setCursor(0, 1);
    lcd.print("Standby       ");

    delay(100);
  }
}
```
